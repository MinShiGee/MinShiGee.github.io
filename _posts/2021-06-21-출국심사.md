최악의 경우가 매우 커서 단순 노가다로는 문제를 해결할 수 없다.

따라서 Parametric Search 알고리즘을 이용해서 시간 값을 mid로 정의하고, 그 mid값 안에 심사를 해결할 수 있는지를 기준으로 값을 구했다.

```
def solution(n, times):
    
    answer = 0
    mx = max(times)
    
    l = 0
    r = (n + 1) * mx
    
    while l <= r:
        mid = (l + r) // 2
        
        t = 0
        for d in times:
            t += mid // d
            if t >= n:
                answer = mid
                r = mid - 1
                break
                
        if t < n:
            l = mid + 1
            
    return answer
```
